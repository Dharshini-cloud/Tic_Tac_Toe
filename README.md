# Tic-Tac-Toe Smart AI Game

A modern, responsive Tic-Tac-Toe web application with two gameplay modes:
-  Two-Player Mode 👥
-  Smart AI Mode 🤖

Built using **HTML, CSS, and JavaScript**, this project demonstrates game logic, UI design, and AI decision-making.

---

## Features

- Two game modes:
  - Player vs Player
  - Player vs AI
- Smart AI opponent:
  - Win if possible
  - Block opponent
  - Take center, corners, or edges strategically
-  Real-time turn indicator
-  Game-over modal with winner display
-  Modern UI with animations and gradients
-  Fully responsive design
-  New Match & Reset options
-  Exit to main menu functionality

---


## Technologies Used

- HTML5
- CSS3 (Flexbox, Grid, Animations)
- JavaScript (DOM Manipulation, Game Logic)
